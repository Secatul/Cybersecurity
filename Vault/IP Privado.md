#redes 
É um número único atribuído para um dispositivo dentro da [[LAN]].

Existem 3 tipos de IP privados

Classe A `10.0.0.0` - `10.255.255.255`
Classe B `172.16.0.0` - `172.31.255.255`
Classe C `192.168.0.0` - `192.168.255.255` 

os mais comuns:
`192.168.1.0` - Classe C
`10.0.0.0` - Classe A
`172.16.0.0` ou  `172.31.0.0` - Classe B

---
## Comando para encontrar IP privado

`ipconfig` ou `ifconfig`
