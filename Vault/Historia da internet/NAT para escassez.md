#historia_da_internet #redes 
Network Address Translation permite que vários dispositivos utilizem um único IP público

Todos os seus dispositivos contém um IP privado:
- Celular
- Televisão
- Microondas
- Geladeira moderna
- Computador

Seriam 5 endereços de IP, o NAT traduz eles para um IP público (Apenas um IP por rede)
